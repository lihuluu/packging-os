# Private Projects

This folder is intentionally ignored in the public repository.

Use it for real packaging projects only in your local workspace:

```text
Workspace/Projects/
  Your Project/
    00_Project_Control/
    01_Brief/
    02_Research/
    03_Design/
    04_Final/
```

Do not commit real client work, supplier details, quotes, proofing records, production files, or final artwork.

For a safe public example, see `examples/demo-tea-project/`.
