## 本轮最值得保留的知识

### 来自 HGT235 特级高山绿茶（本轮新增/修正）
1. **长期合作 ≠ 自动主动沟通**：供应商发现色差但未及时报告 → 需要合同条款约束，不能依赖默契。
2. **视频对色的边界修正**：不是现场跟色的可行替代——屏幕色差是系统性的，凹印多专色产品尤其不适用。起"过程监控"而不是"签样依据"。
3. **合规信息逐字校对**：V1.0 OCR 错误说明印前检查不能只检查"已收到"，需要增加逐字比对源文件环节。
4. **35 天快速交付验证**：业务部门做设计 + Packaging OS 做印后的混合模式可行，6 天完成开案→发厂。
5. **供应商主动报告义务**：DR-008 / RS-007 / PU-007 形成闭环——规则 + 风险信号 + 流程更新打一套。

### 来自 HT2385 猴王·手筑茯茶五年陈（已有，不变）
1. 三天两次方案反转，供应商信任需主动维护。
2. 牛皮纸包裹 + CMYK + 专白，¥2.75/套即达成口粮茶视觉预期。
3. 彩样签样后距上市仅 32 天，合规和外箱必须并行推进。

### 跨项目
1. 电商茶叶包装实际成本 ¥0.95~2.75/件，远低于 ¥5-6 预算上限。

### 来自 苹果黄芪茶
1. 预制铝箔袋套印对位偏差是工序累积公差。
2. 打样外观通过不等于大货无偏差。
3. 应对措施：Brief 阶段明确各工序公差，打样要求累计上限。
