## 证据等级规则

| 字段 | 最低证据等级 | 说明 |
|------|-------------|------|
| 预算值 | C1 | 客户预算或目标成本 |
| 报价值 | C3 | 供应商粗报价及以上 |
| 实际值 | C5 | 打样后修正报价或结算价 |

没有来源和日期的成本不能进入正式成本表。

## 成本证据等级说明

| 等级 | 来源 | 允许输出 |
|------|------|---------|
| C0 | AI 推断 / 行业经验 | 只能输出成本风险 |
| C1 | 客户预算 / 目标成本 | 可判断是否存在预算压力 |
| C2 | 历史项目成本 | 可做参考，但必须标注项目差异 |
| C3 | 供应商粗报价 | 可做初步方案比较 |
| C4 | 正式报价单 | 可做预算判断和供应商比较 |
| C5 | 打样后修正报价 / 结算价 | 可作为真实成本记录 |
