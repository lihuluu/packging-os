## 标准命名

模板产物默认遵循以下命名规范：

- skill id / skill 目录：`kebab-case`
- 项目文档 / 知识文档 / 模板产物：`kebab-case`

标准文件名：

- `project-memory-card.md`
- `project-tracker.md`
- `supplier-brief.md`
- `quotation-comparison.md`
- `packaging-brief.md`
- `brief-decomposition.md`
- `positioning-summary.md`
- `concept-directions.md`
- `visual-direction.md`
- `structure-decision.md`
- `material-finishing-plan.md`
- `ai-concept-brief.md`
- `selection-notes.md`
- `prepress-review.md`
- `asset-register.md`
- `compliance-review.md`
- `sample-acceptance-record.md`
- `proposal-outline.md`
- `project-retrospective.md`
- `knowledge-synthesis.md`

历史命名 `project_tracker.md`、`structure-selection.md` 不再作为新模板文件名使用。
