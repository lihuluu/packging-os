# 印前检查框架

## 目录
- 发厂检查单
- 合规与素材授权检查
- 样品与大货验收
- 打样问题诊断
- 降本检查
- 输出模板

## 发厂检查单

### 文件准备度
- 已确认最终刀模
- 已确认出血与安全区
- 字体已打包或转曲
- 链接已嵌入或打包
- 色彩模式与配置文件正确
- 已检查 overprint 与 knockout
- 条码与法规文案已检查

### 量产准备度
- 材质已确认
- 印刷方式已确认
- 工艺图层已说明
- 色彩参照已定义
- 校样方式已定义
- 渐变、大面积实地、烫金、压纹等敏感工艺已完成实物打样或合同样确认
- 审批负责人已明确

### 合规与授权准备度
- 标签/法规信息已由内部法规、品控或负责人复核
- 宣传语、功效、产地、年份、认证等表述有依据
- 条码、规格、生产信息、保质期、贮存条件和生产日期位置已预留
- 字体、图片、插画、AI 图、认证标志和合作方 Logo 授权状态已记录
- 食品接触材料或内包材说明已向供应商确认

### 样品与大货准备度
- 白样验收状态已记录
- 彩样/合同样验收状态已记录
- 机上样或大货首件验收状态已记录（如适用）
- 大货抽检标准已写清（如适用）
- 让步接收必须写明风险、责任方和审批人

## 合规与素材授权检查

遇到以下情况，应列为阻塞项：
- 法规/标签核心信息缺失或仍由设计侧推测
- 功效、认证、产地、年份等宣传依据不明
- 商用字体、图片、插画、AI 图或合作方标志授权状态不明
- 食品接触材料缺供应商说明或企业内部确认

## 样品与大货验收

| 阶段 | 必看重点 | 不通过示例 |
| --- | --- | --- |
| 白样 | 结构、尺寸、折线、开合、内托、装箱 | 爆线、内托松、开合卡、装箱不稳 |
| 彩样/合同样 | 色差、材质、印刷、烫金/UV/压纹、条码 | 明显色偏、工艺偏位、条码不可读 |
| 机上样 | 上机色彩、套印、专色、批量稳定性 | 套印不稳、实地不均、专色偏差大 |
| 大货首件 | 与签样一致性、批量缺陷 | 首件即偏离签样 |
| 大货抽检 | 外观、数量、批次、运输损伤 | 批量擦花、溢胶、数量不符 |

## 打样问题诊断

常见问题类型：
- 色偏
- 套印不准
- 折线爆裂
- 覆膜或烫金失败
- 擦花
- 附着力不足
- 变形或结构翘曲

每个问题都要写：
- 现象
- 可能原因
- 建议检查
- 立即修复
- 预防性修改

## 降本检查

- 优先减复杂度，不先牺牲核心可读性
- 先简化工艺，再考虑是否要改材质等级
- 一旦结构变复杂，要重算装配人工
- 先确认成本到底是色数还是特殊工艺在拉高
