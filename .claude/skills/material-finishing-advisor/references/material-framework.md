## 概念到材质翻译

| 概念关键词 | 材质特征 | 工艺倾向 |
| --- | --- | --- |
| 极简 / 留白 | 纯色纸张、无涂层、细腻触感 | 压凹、局部击凸、少量烫金 |
| 质感 / 手工 | 纹理纸、棉纸、纤维感材质 | 压纹、烫金、贴标 |
| 科技 / 现代 | 光面纸、金属感材质、干净表面 | UV、烫银、专色高对比 |
| 自然 / 环保 | 再生纸、FSC 纸、低加工材质 | 大豆油墨、少覆膜、低复合工艺 |
| 礼赠 / 仪式感 | 厚板、包裱结构、触感明确的面材 | 烫金、压纹、磁吸、内托细节 |
| 克制 / 男性化 | 深色调纸材、低反光表面、稳定纹理 | 哑金、击凸、少量专色、减少高亮工艺 |
| 传统 / 文化感 | 棉纸、特种纹理纸、带纤维感或自然颗粒 | 压凹、烫金、专色、局部图文肌理 |

## 翻译规则

- 先把概念关键词翻成 `材质特征`，再决定工艺，不要反过来先堆工艺。
- 触感目标、视觉目标、环保目标分开写，避免一句“高级感”同时承担三个任务。
- 如果同一个概念同时包含冲突词，例如“极简 + 奢华”或“环保 + 高反光金属感”，必须单独写出取舍。
- 如果概念来自概念层文档，优先引用概念原文关键词，不要在材料层重新发明一套表达。

## 评估维度

- 保护性与耐用性
- 触感信号
- 视觉信号
- 印刷兼容性
- 工艺兼容性
- 交期
- 成本压力
- 回收或处置影响

## 正式锁定前必须验证

- 供应商是否真实支持该纸材和工艺组合
- MOQ 是否支撑该方案的单价结构
- 打样方式是否能真实反映关键工艺效果
- 结构与材质厚度是否匹配
- 环保主张是否会被覆膜、复合或高损耗工艺削弱
- 如果概念高度依赖触感，是否需要先做白样或材质小样确认

## 环保提示

- 当前工艺组合是否仍支持环保主张？
- 是否能先做减材，再谈高级工艺？
- 所在区域是否有回收或标识规定？
- 材质选择是否反过来限制了印刷或工艺？
