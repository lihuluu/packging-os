### 样本类型说明

| 类型 | 定义 | 用途 |
|------|------|------|
| 直接竞品 | 同品类、同价格带、同渠道 | 定位差异化的直接参照 |
| 渠道竞品 | 不同品类但同一货架/渠道 | 货架表现和购买场景的参照 |
| 审美参考 | 视觉语言/调性契合但品类无关 | 视觉方向的灵感来源，不作为定位依据 |
| 跨品类借鉴 | 解决了类似问题的其他品类 | 方法论迁移，不直接对比 |
| 反例样本 | 明确不想成为的样子 | 边界界定，排除方向 |

### 证据等级说明

| 等级 | 含义 |
|------|------|
| E0 | 无证据，纯假设 |
| E1 | 行业经验推断 |
| E2 | 二手信息（官网、报告、电商截图） |
| E3 | 间接验证（实地走访、客户口头确认、销售反馈） |
| E4 | 直接验证（实物对比、货架实拍、用户测试、销售数据） |
