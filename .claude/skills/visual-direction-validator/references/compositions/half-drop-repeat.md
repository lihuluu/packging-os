# half-drop-repeat（半降重复图案）

经典纺织风格半降重复图案，用于全面覆盖包装表面。

## 结构

- 主体图案在规则网格中重复
- 相邻列（或行）偏移 50%——即"半降"
- 图案在水平和垂直两个方向无缝拼接
- 无明显的起止点

## 半降的原理

```
A 列：  [图案]         [图案]
B 列：      [图案]         [图案]   （向下偏移 50%）
```

这种结构比直排重复产生更有机、更流动的感觉。

## 尺度考量

- 大图案（重复单元的 30-40%）：醒目，宣言式图案
- 中等图案（15-25%）：经典，均衡
- 小图案（5-10%）：精细，纺织感，退居辅助地位

## 适合场景

礼品包装纸、奢华礼盒、时尚品牌联名、有季节变体的产品。重复图案创造高端的表面设计，提升产品感知价值。

## 与 `pattern-allover` 的区别

`pattern-allover` 是主题（重复什么内容）——`half-drop-repeat` 是构图结构（如何重复）。两者相互配合使用。

## AI 提示词中的构图说明

指定：无缝半降重复图案，[图案描述]以半降纺织网格形式重复，均匀覆盖表面，图案在所有方向无缝拼接，[大/中/小]重复单元尺寸。
