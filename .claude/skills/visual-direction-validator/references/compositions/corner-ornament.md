# corner-ornament（角落装饰）

装饰性插画点缀在角落或边缘，为中央留白区域构建框架。

## 结构

- 插画集中于 2 或 4 个角落（或上下边缘）
- 中央空白区域留给产品名称和核心文案
- 插画作为建筑式框架，而非焦点内容
- 可对称或不对称布置

## 变体形式

| 变体 | 说明 |
|-----------|-------------|
| 四角 | 四个角各有相同的装饰元素 |
| 对角线双角 | 左上 + 右下（或反之） |
| 上下横条 | 插画仅跨越上下边缘 |
| 角落 + 中心 | 小型角落装饰 + 单个中心点缀 |

## 视觉元素

- 植物小枝、藤蔓、枝梢
- 装饰性卷草纹、花饰
- 小型角色或纹样浮雕
- 几何角落内嵌图案

## 适合场景

礼品盒、茶叶罐、糖果包装、经典食品。营造正式而考究的印象——如同精品信纸。传递传统、用心、礼仪感。

## AI 提示词中的构图说明

指定：装饰性插画元素置于角落（说明：左上和右下，或四角均有），中央区域留白用于产品信息，植物/装饰性角落图案，整体对称感，精致细腻。

- `botanicals` + `botanical-engraving`：经典茶饮或果酱标签
- `cultural-motif` + `art-nouveau`：传统礼品盒
- `botanicals` + `vintage-etching`：精品杂货、糖果
