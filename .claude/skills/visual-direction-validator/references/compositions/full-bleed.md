# full-bleed（满版铺陈）

插画铺满整个包装面——没有"标签中的标签"，只有纯粹的插画。

## 结构

- 插画延伸至边缘（含出血位）
- 无正式边框或框架
- 产品名称悬浮于插画之上（独立文字图层）
- 背景即插画本身

## 视觉元素

- 强烈的氛围感背景（天空、水面、田野、抽象底色）
- 主体自然地融入或置身于背景之中
- 层次与景深：前景 / 中景 / 背景分区
- 为产品名称预留清晰的版面区域（通常为色调较浅的区域）

## 文字版面预留区

设计插画前，确定产品名称将放置在哪里：
- 通常是上三分之一（清澈的天空，浅色背景）
- 或底部横条（地面，阴影，低饱和区域）
- 必须在提示词中说明这一要求

## 适合场景

现代快消品、精酿啤酒罐、自立袋、生活方式品牌，任何想要沉浸感与视觉冲击力的品牌。传递的信号："我们足够自信，让插画独当一面。"

## AI 提示词中的构图说明

指定：边到边满版插画，无边框，主体自然融入场景，[上方/下方]区域相对留白（色调较浅或细节较少，供产品名称使用），满版出血插画。

## 推荐组合

- `landscape-geography` + `painterly-expressive`：精酿烈酒、精品咖啡
- `brand-mascot` + `gouache-flat`：趣味零食品牌
- `abstract-botanical` + `watercolor-wash`：现代护肤品
- `wildlife-nature` + `linocut-relief`：精酿啤酒、环保品牌
