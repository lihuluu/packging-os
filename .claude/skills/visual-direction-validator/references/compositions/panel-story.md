# panel-story（分格叙事）

多个插画格子叙述连续故事或多角度呈现。

## 结构

- 2-6 个独立插画格（类似漫画分镜或故事板）
- 每个格子包含一个独立的插画场景
- 格子从左到右或从上到下流动叙事
- 可以是等大的网格，也可以是大小各异的格子

## 故事类型

| 故事 | 分格顺序 |
|-------|---------------|
| 产地之旅 | 农场 → 加工 → 产品 → 享用 |
| 一天生活 | 早晨 / 正午 / 傍晚场景 |
| 四季故事 | 春 / 夏 / 秋 / 冬 |
| 工艺流程 | 种植 → 收获 → 精制 → 品鉴 |
| 多变体展示 | 不同口味/变体各有专属插画格 |

## 设计原则

- 每个格子必须能独立阅读（考虑小标签尺寸）
- 所有格子保持统一的风格与色彩方案
- 阅读顺序的视觉逻辑要清晰
- 格子边界可以是建筑性的（规则线）或柔和的（渐变融合）

## 适合场景

精酿啤酒罐（背面）、高端巧克力棒、多面板礼品盒、有故事可讲的手工坊品牌。分格叙事邀请消费者花时间与包装互动。

## AI 提示词中的构图说明

指定：[数量]格连续插画，格子[水平/垂直]排列，每格描绘[场景说明]，所有格子风格统一，格子之间有清晰分隔，设计为从左至右阅读。

## 推荐组合

- `landscape-geography` + `retro-lithograph`：精酿啤酒产地故事
- `food-ingredients` + `folk-naive`：手工食品工艺故事
- `lifestyle-scene` + `painterly-expressive`：高端品牌叙事
