# border-frame（边框装饰）

装饰性插画边框围绕整个标签边缘，将中央内容框起来。

## 结构

- 插画沿四边边缘连续延伸，形成装饰边框
- 内部留白区域用于产品名称、广告语、文案
- 边框在四角处有装饰性衔接节点
- 可以是细边框（精致优雅）或宽边框（插画主导）

## 边框类型

| 类型 | 特征 |
|------|-----------|
| 植物藤蔓 | 连续植物生长环绕边缘 |
| 几何装饰 | 沿边框重复的几何单元 |
| 飘带横幅 | 建筑式飘带元素 |
| 雕版线饰 | 精细线条装饰纹，角落有装饰元素 |
| 民间纹样 | 密集重复的民俗纹样边框 |

## 宽度参考

- 细边框（标签宽度的 5-8%）：优雅克制，文案主导
- 中等边框（10-15%）：均衡，传统标签感
- 宽边框（20-25%）：插画主导，沉浸式传统风格

## 适合场景

果酱、蜜饯、葡萄酒、传统食品、礼品的经典标签。边框装饰是传统、工艺感与用心细节的强烈信号。
