# modern-flat（现代扁平）

当代简洁扁平矢量插画——无阴影、无质感、纯粹形状与色彩。

## 视觉特征

- 完美干净的平色填充
- 无阴影、无渐变、无质感
- 主体简化为本质几何形与有机形
- 任何尺寸下都保持极高的视觉清晰度
- 当代插画作品集美学
- 受当代编辑插画、DTC 品牌设计影响

## 色彩方案

- 精选和谐色板（通常 4-6 种颜色）
- 常见组合：一个暖色 + 一个冷色 + 一个强调色 + 中性色
- 可以是明亮/鲜艳，也可以是柔和/低饱和——取决于品牌
- 无金属感、无质感——纯粹平面数码色彩

## 形态语言

- 自然主题用有机形态（植物草本、食材）
- 结构性主题用几何形态
- 简化但不粗糙——是经过思考的提炼
- 无轮廓线，或统一线重轮廓（选其一，保持统一）

## 排版搭配

- 干净几何无衬线：Circular、Apercu、Neue Haas
- 中等字重，视觉间距
- 与插画的精准感相匹配

## 印刷注意事项

- CMYK 还原效果极佳——干净平色转换良好
- 完美适配任何尺寸
- 非常适合数字原生品牌、DTC、丝印
- 适配任何底材——表面质感不影响效果

## 最佳搭配

- 主题：`abstract-botanical`、`food-ingredients`、`brand-mascot`
- 构图：`floating-elements`、`full-bleed`、`central-medallion`

## 提示词关键词

现代扁平矢量插画，干净几何形态，无阴影无质感无渐变，当代编辑插画风格，简化有机形态，纯平色填充，DTC 品牌插画美学，精准干净，无轮廓线风格，当代现代插画
