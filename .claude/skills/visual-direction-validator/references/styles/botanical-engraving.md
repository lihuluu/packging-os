# botanical-engraving（植物铜版画）

维多利亚植物学插画风格——精细线条铜版画，精准细节。

## 视觉特征

- 精细、精准的线条，传承铜版画雕刻传统
- 交叉排线与点刻用于表现阴影与体积
- 黑色油墨在奶油色/白色纸上，或加入平涂水彩的彩色版本
- 极致的植物细节：叶脉、花瓣数量、雄蕊的精准呈现
- 科学插画传统——奥杜邦/雷杜德的遗产

## 色彩方案

- 经典版：奶油底（#FEFAF0）或做旧白底（#F5F0E8）上的黑色墨线
- 彩色版：精细雕刻线条内填涂柔和植物学水彩
- 点缀色：暖褐色（#8B6548）、灰绿色（#7A9E7E）、浅砖红（#C8846A）
- 金色：高端版本可加入少量金线点缀

## 线条质感

- 极细精准的线条（等效 0.5-1pt）
- 线条粗细变化表现深度层次
- 平行排线表现阴影，点刻表现纹理
- 无松散或随意的笔触——每条线都是有意为之

## 排版搭配

- 衬线字体：Garamond、Caslon、Didot、Bodoni
- 产品名称用大写字母加宽字距
- 植物拉丁学名用斜体

## 印刷注意事项

- 精细线条印刷至少需要 600 DPI
- 避免：最终印刷尺寸下小于 0.3mm 的极细线条
- 胶印和数码印刷均可很好还原
- 压凹工艺与此风格极为契合
