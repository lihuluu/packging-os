# vintage-etching（古典蚀刻）

精细交叉线蚀刻风格——古墨质感、雕版印刷、做旧纸张气质。

## 视觉特征

- 精细交叉线条积累色调与体积
- 凹版印刷工艺传统（铜版蚀刻）
- 略显泛黄的做旧纸张质感
- 主体完全通过线条密度变化来表现
- 精致、考究、耗时的匠心感
- 参考：阿尔布雷希特·丢勒、伦勃朗的蚀刻版画、古代自然史插画

## 色彩方案

- 单色版：黑色/深褐油墨（#2A1A0A）在做旧奶油纸（#F5EDD8）或泛黄白纸（#FAEBD7）上
- 深褐色调：暖棕色油墨（#6B4226）营造古典感
- 偶尔：为单色蚀刻添加极浅的水彩晕染（彩色版）
- 做旧纸张质感：全程轻微暖黄色调

## 线条技法

- 平行排线：线条紧密排列表现较深色调
- 交叉排线：垂直方向叠加排线，制造更深的阴影
- 点刻：用于极细腻的质感表现
- 划亮：较亮区域的线条更稀疏、间距更大
- 线条方向顺应形体（轮廓排线增强立体感）

## 与 botanical-engraving 的区别

| botanical-engraving（植物铜版画） | vintage-etching（古典蚀刻） |
|---|---|
| 以植物为主要主题 | 主题不限 |
| 常有彩色版本 | 通常为单色 |
| 科学标本板传统 | 纯艺术蚀刻传统 |
| 干净白色底 | 做旧/泛黄纸底 |

## 印刷注意事项

- 精细线条需要 600+ DPI 分辨率
- 暖褐色调在 CMYK 中还原效果极佳
- 做旧纸张效果需要细致的印前处理
- 凸版印刷与活版印刷是天然工艺搭档

## 最佳搭配

- 主题：`wildlife-nature`、`landscape-geography`、`botanicals`
- 构图：`central-medallion`、`floating-elements`、`border-frame`
