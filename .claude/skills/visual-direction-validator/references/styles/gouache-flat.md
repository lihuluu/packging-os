# gouache-flat（水粉平涂）

大胆厚涂水粉平涂画风——鲜艳不透明的色块，干净边缘，当代平面感。

## 视觉特征

- 不透明、哑光的色彩——无透明感
- 干净的平涂色域，无或极少渐变
- 强烈的色彩对比与高饱和度
- 平面海报感、图形感
- 大胆清晰的形状，边缘锐利
- 形态简化——既非写实，也非卡通

## 色彩方案

- 鲜明不透明：电光蓝、朱红、金黄、森林绿
- 通常为高饱和主色调
- 或：精炼的有限色板（2-3 个大胆颜色 + 中性色）
- 哑光表面质感——无光泽或反光

## 绘画技法

- 平涂：纯色、均匀覆盖的色域
- 无可见笔触（与笔触表现风格不同）
- 形状由颜色变化定义，而非轮廓线
- 极简阴影：简化为更深的平调

## 排版搭配

- 粗重几何无衬线：Futura、Montserrat Bold、DIN
- 强烈的图形化字体与大胆插画相配
- 全大写与此风格效果极佳

## 印刷注意事项

- CMYK 还原效果极佳——干净的平色转换良好
- 高对比度在小尺寸印刷中依然清晰
- 非常适合柔版印刷（零食袋等）
- 在涂布纸和非涂布纸上均表现良好

## 最佳搭配

- 主题：`food-ingredients`、`brand-mascot`、`botanicals`
- 构图：`full-bleed`、`central-medallion`、`floating-elements`
