# art-nouveau（新艺术风格）

流动有机的新艺术主义风格——曲线形态、自然灵感的装饰、优雅蜿蜒的线条。

## 视觉特征

- 从自然形态中提炼的流动曲线
- 鞭打曲线——新艺术主义标志性的 S 形和 C 形曲线
- 自然形态（植物、藤蔓、花卉、昆虫、女性发丝）作为装饰结构
- 高度装饰性——装饰本身即目的
- 对称中带有有机变化
- 参考艺术家：穆夏（Mucha）、克里姆特（Klimt）、比亚兹莱（Beardsley）

## 色彩方案

- 宝石色调搭配金色：深青（#2B5F6B）、酒红（#722F37）、森林绿（#3D6B47）
- 金色点缀：#C9A227 或金属金
- 柔橄榄绿与灰绿色
- 深紫色/堇色
- 奶油色或象牙白背景
- 绝不用纯色或过度饱和——始终保持精致感

## 装饰元素

- 风格化花卉边框与框架
- 流动的藤蔓与卷须结构
- 飘逸发丝的女性形象作为纹样
- 孔雀羽毛、蜻蜓翅膀
- 相互咬合的有机几何形

## 排版搭配

- 新艺术主义展示字体：Arnold Böcklin、青春风格（Jugendstil）灵感字体
- 蜿蜒的字形，有机衬线
- 产品名称用全大写紧字距

## 印刷注意事项

- 精细曲线细节需要高分辨率（600 DPI 以上）
- 金色颜色需要专色金属油墨才能真实呈现
- 烫金与压凹工艺是天然搭档
- 在中大尺寸标签上效果最佳（小尺寸下细节会丢失）

## 最佳搭配

- 主题：`botanicals`、`cultural-motif`、`pattern-allover`
- 构图：`central-medallion`、`border-frame`、`half-drop-repeat`
