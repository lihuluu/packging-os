# lifestyle-scene（生活场景）

生活场景主题：使用产品的人物场景、生活瞬间、日常活动。

## 描绘内容

- 使用瞬间：有人正在享用产品（喝茶、涂护肤品）
- 社交场景：朋友共享美食，家庭聚会
- 活动语境：带着产品徒步，晨间日常
- 向往瞬间：使用产品后的感受与状态
- 抽象生活方式：风景中的剪影，置身环境中的人物

## 场景构图原则

- 营造清晰的氛围与时间感（晨光、黄金时段、温馨傍晚）
- 人物形象应风格化，非写实（避免恐怖谷效应）
- 环境与人物同样重要，共同讲述故事
- 将产品（或品类暗示）自然地融入场景
- 避免过于特定的文化指向（限制市场），除非面向地域性品牌

## 情绪光谱

| 情绪 | 视觉处理 |
|------|-----------------|
| 温馨 / 亲密 | 暖光，小型居家场景，柔和质感 |
| 活跃 / 充满能量 | 鲜明色彩，动态姿势，开阔户外场景 |
| 向往 / 奢华 | 干净线条，精致场景，人物克制 |
| 社群 / 共享 | 多人物，温暖热闹的场景 |

## 最佳搭配

- 构图：`full-bleed`、`panel-story`、`wrap-around-360`
- 风格：`painterly-expressive`、`retro-lithograph`、`modern-flat`、`gouache-flat`

## 适合品类

咖啡、高端烈酒、运动营养、旅行/探险品牌、社交饮料
