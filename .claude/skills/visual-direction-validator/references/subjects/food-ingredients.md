# food-ingredients（食材食品）

食材食品主题：原料食材、新鲜农产品以及能激发食欲的食物呈现。

## 描绘内容

- 原料食材：新鲜蔬菜、水果、谷物、坚果、香料
- 加工元素：切开的柑橘、敲开的鸡蛋、倾倒的蜂蜜、成堆的香料
- 食欲激发信号：蒸汽、淋汁、横截面展示、新鲜感传达
- 艺术排列：食材的松散散落或结构化构图
- 主角食材：一种食材被突出强调、重点展示

## 插画处理方式

- 通过色彩和状态强调新鲜感与品质感
- 运用食欲激发信号：光泽感、蒸汽上升、鲜亮色彩
- 横截面视角展示内部品质（柑橘切片、番茄剖面）
- 松散散落传达丰盛感；单个主角食材传达高端感

## 情绪光谱

| 情绪 | 表现方式 |
|------|-----------|
| 新鲜 / 活力 | 高饱和明亮色彩，轮廓清晰 |
| 乡村 / 手工坊 | 大地色调，粗粝质感，不完美形态 |
| 高端 / 精致 | 单一主角食材，精细表现 |
| 趣味 / 活泼 | 卡通风格，夸张的新鲜感信号 |

## 最佳搭配

- 构图：`central-medallion`、`floating-elements`、`full-bleed`
- 风格：`watercolor-wash`、`gouache-flat`、`painterly-expressive`、`folk-naive`

## 适合品类

酱料、调味品、零食、新鲜农产品、果汁、果酱、手工食品、烹饪原料
