# brand-mascot（品牌角色）

品牌角色或吉祥物作为主要插画主题。

## 描绘内容

- 拟人化角色（被赋予个性的动物、物体、食品）
- 人物角色（品牌创始人、文化原型、生活方式人物）
- 奇幻生物（友好的怪兽、为品牌改编的神话生物）
- 物品角色（产品本身被赋予个性）

## 角色设计原则

- 轮廓清晰——在小尺寸包装上也能辨认
- 表情丰富——一眼传达品牌个性
- 风格统一——可在不同包装变体中复用
- 动态姿势——角色在做某件事，而不是干立
- 避免：过于复杂的细节（小尺寸印刷后会消失）

## 个性与形态的对应关系

| 品牌个性 | 角色类型 |
|-------------------|---------------|
| 亲切 / 趣味 | 圆润造型，暖色调，大笑容 |
| 高端 / 精致 | 干净线条，克制表情，优雅姿势 |
| 天然 / 环保 | 动物角色，与自然相连，柔和色调 |
| 传统 / 值得信赖 | 经典比例，庄重，历经岁月的质感 |
| 冒险 / 活力 | 动感姿势，大胆用色，充满能量 |

## 最佳搭配

- 构图：`central-medallion`、`full-bleed`、`panel-story`
- 风格：`children-storybook`、`gouache-flat`、`folk-naive`、`retro-lithograph`

## 适合品类

儿童食品、宠物食品、零食、谷物早餐、精酿啤酒、个性鲜明的手工品牌
