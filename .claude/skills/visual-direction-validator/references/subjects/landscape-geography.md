# landscape-geography（地域风景）

产地风景与地理身份作为主要插画主题。

## 描绘内容

- 农场 / 产地场景：连绵起伏的田野、梯田山丘、葡萄园列阵
- 地域地标：山脉轮廓、海岸线、河谷
- 农业劳作瞬间：收获活动、耕作场景、加工过程
- 季节性风景：樱花春日、丰收金秋、冬日白雪
- 风土元素：土壤、阳光、雨水、微气候的视觉暗示
- 地图式呈现：风格化地图展示产地路线

## 叙事力量

这个主题讲述的故事是："这个产品来自某个真实的、特定的地方。"

| 故事类型 | 视觉处理 |
|---|---|
| 单一农场 / 庄园 | 亲密场景，可辨认的农场建筑 |
| 地域身份 | 宽阔风景，捕捉一个地区的精髓 |
| 产地之旅 | 地图式或路线式插画 |
| 季节性收获 | 丰收高峰时节的劳作场景 |

## 插画处理方式

- 包含具体的地理标志（山形、河湾、梯田）
- 注重尺度感：风景中的微小人物强化风土叙事
- 天气与光线是故事的一部分（晨雾=神秘感，黄金时段=温暖）
- 避免泛泛的"乡村风光"——必须是无可取代的特定产地

## 最佳搭配

- 构图：`wrap-around-360`、`full-bleed`、`panel-story`
- 风格：`retro-lithograph`、`painterly-expressive`、`linocut-relief`、`watercolor-wash`

## 适合品类

葡萄酒、威士忌、精酿啤酒、精品咖啡、手工奶酪、特级初榨橄榄油、地域茶
