# pattern-allover（满版图案）

重复图案元素覆盖包装整个表面。

## 描绘内容

- 植物重复图案：花卉、叶片、藤蔓全面铺满
- 几何重复图案：源自品牌语言的结构性重复形状
- 角色散落图案：品牌吉祥物在表面重复出现
- 传统纹样图案：经典重复纹样的品牌化改编
- 叙事密铺图案：无缝拼接的复杂插画场景

## 图案重复类型

| 类型 | 结构 | 效果 |
|------|-----------|--------|
| 直排重复 | 元素对齐成行排列 | 结构感，几何感，正式 |
| 半降重复 | 交替行偏移 50% | 经典纺织感，更有机 |
| 砖形重复 | 交替行水平偏移 50% | 平衡，传统 |
| 菱形/格架 | 斜向网格结构 | 优雅，花园风格 |
| 涡纹 | S 曲线相互咬合 | 传统，挂毯质感 |
| 随机散落 | 无严格网格，自然分布 | 趣味，现代，丰盛感 |

## 包装图案设计原则

- 用于环绕包装时，图案必须无缝拼接
- 尺寸：在实际包装尺寸下测试效果（图案缩小后往往大不相同）
- 考虑"图底反转"：能否在浅底上用深色，也能在深底上用浅色？
- 为标签文字版面预留留白区域
- 避免图案过于繁复影响产品名称的可读性

## 最佳搭配

- 构图：`half-drop-repeat`、`full-bleed`
- 风格：`art-nouveau`、`folk-naive`、`botanical-engraving`、`modern-flat`

## 适合品类

礼品盒、糖果、花草茶、时尚联名、化妆品套装、奢华礼品包装
