# abstract-botanical（抽象植物）

从自然中提炼的抽象有机形态——不是写实的植物描绘，而是植物启发的形状。

## 描绘内容

- 植物形状简化为本质几何形：花瓣→椭圆，叶片→水滴形
- 源自植物色素的色彩场域（叶绿素绿、花瓣粉、树皮棕）
- 有机曲线与仿生形态
- 细胞结构或微观植物视角
- 暗示生长、运动、生命能量的流动形态

## 与 `botanicals` 的区别

| botanicals（植物草本） | abstract-botanical（抽象植物） |
|---|---|
| 可识别的物种 | 无法辨认但源于自然 |
| 具象 / 写实 | 抽象 / 非具象 |
| 科学准确性可能 | 情感共鸣优先于准确性 |
| 传统插画风格 | 当代艺术方向 |

## 适用场景

适合以下品牌：
- 希望传达"天然"信号，但不想用写实植物插画的品牌
- 已超越植物俗套的现代/高端护肤品牌
- 科学感与天然感兼顾的保健品品牌
- 希望以自然为灵感但与众不同的包装设计

## 最佳搭配

- 构图：`floating-elements`、`full-bleed`
- 风格：`japanese-minimalist`、`modern-flat`、`watercolor-wash`

## 适合品类

高端护肤、健康保健品、功能性饮料、现代美妆、轻生活方式品牌
