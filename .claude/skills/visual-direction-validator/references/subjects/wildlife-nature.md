# wildlife-nature（野生动物自然）

野生动物与自然主题：动物、鸟类、昆虫以及自然景观。

## 描绘内容

- 动物在自然栖息地（熊、狐狸、鹿、野兔、狼）
- 具有象征意义的鸟类（丹顶鹤、猫头鹰、雄鹰、燕子、蜜蜂）
- 昆虫主角（蜜蜂、蝴蝶、蜻蜓）
- 水生生物（三文鱼、锦鲤、鲸鱼——适合相关产品）
- 动物动态（采蜜的蜜蜂、飞翔的鸟儿、觅食的鹿）

## 动物象征意义

| 动物 | 常见联想 |
|--------|-------------------|
| 熊 | 力量、野性、加拿大/北欧气质 |
| 狐狸 | 机智、手工坊精神、匠人品质 |
| 鹿 / 雄鹿 | 自然、纯净、苏格兰高地 |
| 蜜蜂 | 手作、勤劳、蜂蜜、天然 |
| 丹顶鹤 | 长寿、优雅、日本/中国文化 |
| 狼 | 野性、高端、群体精神 |
| 野兔 | 速度、自然、英式趣味 |
| 锦鲤 | 好运、日式美学 |

## 插画处理方式

- 将动物置于自然环境中（而非孤立在白色背景，除非有意为之）
- 将动物作为故事叙述工具，而非单纯的 Logo 标记
- 季节语境丰富叙事（雪景=冬日，花开=春天）
- 与植物草本结合，构建更丰富的自然世界场景

## 最佳搭配

- 构图：`central-medallion`、`full-bleed`、`wrap-around-360`
- 风格：`botanical-engraving`、`vintage-etching`、`linocut-relief`、`watercolor-wash`

## 适合品类

蜂蜜、三文鱼、野味肉类、苏格兰/北欧烈酒、精酿啤酒、茶饮、健康产品
