# botanicals（植物草本）

植物草本主题：以植株、草药、花卉、叶片和枝条作为主要插画内容。

## 描绘内容

- 草药：与产品相关的具体植物种类（洋甘菊、薰衣草、薄荷、人参）
- 花卉：完整花朵、花蕾、各状态的花瓣
- 叶片：详细的植物学叶片研究，包括茎干、叶脉
- 枝条：植物的骨架结构、浆果、种子
- 根系：草本/药用产品适合展示根部系统
- 横截面：切开的水果或植物标本

## 插画处理方式

- 准确描绘指定物种（传承植物学插画的严谨传统）
- 展示完整的植物生命周期元素：花朵 + 叶片 + 茎干组合呈现
- 通过大小变化制造视觉层次（大焦点花朵 + 小细节元素）
- 注重精细细节：叶脉、花瓣纹理、雄蕊

## 情绪光谱

| 情绪 | 表现方式 |
|------|-----------|
| 科学 / 准确 | 精确的物种描绘，植物标本板风格 |
| 浪漫 / 柔和 | 松散排列，花朵相互叠压 |
| 民间 / 手工坊 | 图案化的简化造型，精简细节 |
| 极简 / 现代 | 单株标本，漂浮在白色背景上 |

## 最佳搭配

- 构图：`central-medallion`、`floating-elements`、`border-frame`
- 风格：`botanical-engraving`、`watercolor-wash`、`art-nouveau`、`folk-naive`

## 适合品类

茶饮、草本保健品、天然护肤、香薰、植物烈酒、草本酱料
