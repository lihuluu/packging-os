# 成本依据标准

## 核心原则

- AI 不得自行给出真实成本、单价区间或"能控制在 xx 元"的判断。
- 没有供应商报价、历史项目数据、客户预算或结算记录时，只能判断成本压力。
- 所有成本判断必须标注证据等级、来源、日期和口径。

## 成本证据等级

| 等级 | 来源 | 允许输出 |
|------|------|---------|
| C0 | AI 推断 / 行业经验 | 只能输出成本风险 |
| C1 | 客户预算 / 目标成本 | 可判断是否存在预算压力 |
| C2 | 历史项目成本 | 可做参考，但必须标注项目差异 |
| C3 | 供应商粗报价 | 可做初步方案比较 |
| C4 | 正式报价单 | 可做预算判断和供应商比较 |
| C5 | 打样后修正报价 / 结算价 | 可作为真实成本记录 |

## 禁止表达

- 成本低 / 很便宜
- 成本可控
- 能控制在 xx 元
- 留白更省钱
- 抽屉盒一定更贵
- 大概 xx 元左右

## 推荐表达

- 相对工艺复杂度较低，真实成本需报价确认。[C0]
- 当前只有预算目标，不能判断真实单价。[C1]
- 该方案存在成本压力，主要来自结构复杂度 / 特种纸 / 后道工艺 / MOQ。[C0]
- 基于供应商粗报价，初步单价约为 xx 元，需确认是否含税含运含刀模。[C3]

## 适用范围

凡涉及成本、预算、报价、工艺单价、结构成本、量产可行性的输出，必须遵守本标准。
没有 C3 及以上证据时，不得输出真实成本判断。
没有 MOQ、数量区间、材质规格、工艺口径时，不得比较供应商价格。
