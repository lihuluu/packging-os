# 样品与大货验收标准

## 核心原则

- 真实包装项目不能只以效果图或设计稿作为完成标准；白样、彩样和大货都需要可被共同确认的验收标准。
- 验收标准应在打样 brief 阶段写给供应商，而不是等样品出问题后再补。
- 未通过白样或彩样的项目，不应标记为可进入大货；未完成大货验收记录的项目，不应标记为量产闭环。

## 样品阶段定义

| 阶段 | 目的 | 主要看什么 | 常见结论 |
| --- | --- | --- | --- |
| 白样 | 验证结构、尺寸、装配、开合和运输保护 | 刀模、折线、内托、松紧、装箱 | 结构通过 / 需改刀模 / 需换结构 |
| 数码样 | 快速看图文、比例、色彩方向 | 信息层级、图文位置、基础色感 | 可继续 / 需改设计稿 |
| 彩样/合同样 | 验证接近量产的印刷、材质、工艺 | 色差、纸张、烫金、UV、压纹、覆膜 | 可签样 / 需二次打样 |
| 机上样 | 验证真实上机效果 | 专色、实地、套印、批量稳定性 | 可开印 / 需调机 |
| 大货首件 | 量产启动前确认 | 与签样一致性、批量缺陷 | 可继续生产 / 暂停修正 |
| 大货抽检 | 收货验收 | 色差、尺寸、装配、外观、数量 | 接收 / 返工 / 让步接收 |

## 验收维度

| 维度 | 检查项 |
| --- | --- |
| 结构 | 尺寸、折线、开合、内托固定、装箱适配、跌落/挤压风险 |
| 外观 | 色差、污点、擦花、压痕、毛边、溢胶、覆膜气泡 |
| 印刷 | 套印、糊字、断线、渐变、实地均匀度、条码可读性 |
| 工艺 | 烫金掉金、UV 偏位、压凸压凹深浅、模切偏移 |
| 材料 | 纸张手感、挺度、气味、食品接触材料说明 |
| 信息 | 法规文案、条码、规格、生产信息、可变信息区 |
| 数量 | 交付数量、损耗、备品、批次标识 |

## 验收状态

| 状态 | 含义 |
| --- | --- |
| `未开始` | 尚未打样或未收到大货 |
| `待验收` | 已收到样品或大货，尚未形成结论 |
| `通过` | 满足当前阶段验收要求 |
| `有条件通过` | 存在轻微问题，但不影响当前阶段推进，需记录让步原因 |
| `不通过` | 存在阻塞问题，必须返工或重打 |

## 输出要求

每次样品或大货检查必须记录：

- 样品阶段
- 样品版本/批次
- 供应商
- 验收日期
- 验收人
- 对照标准或签样
- 问题清单
- 结论状态
- 下一步动作
